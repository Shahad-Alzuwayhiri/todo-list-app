# 📝 To-Do List App

![Project Banner](images/ToDo%20List%20Project.png)

A simple and interactive **To-Do List application** built using HTML, CSS, and JavaScript.
It helps users manage daily tasks easily with a clean and responsive UI.

---

## 🚀 Live Demo

🔗 [View Live Project](https://shahad-alzuwayhiri.github.io/todo-list-app/)

---

## 🚀 Features

* Add new tasks
* Mark tasks as completed ✔️
* Delete tasks ❌
* Save tasks using Local Storage
* Tasks remain after refreshing the page

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)
* Local Storage API

---

## 💡 How It Works

1. User types a task in the input field
2. Clicks **Add** or presses Enter
3. Task appears in the list
4. Click task → mark as completed
5. Click ❌ → delete task
6. Data is saved automatically in browser
